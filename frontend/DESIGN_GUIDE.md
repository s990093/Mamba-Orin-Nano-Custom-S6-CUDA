# Apple Design System - 快速指南

## 🎨 設計特色

### Liquid Glass (液態玻璃)
- 半透明卡片，背景模糊
- 光影反射，層次感強
- 互動時的發光效果

### 動畫效果
- 訊息滑入動畫 (左/右彈跳)
- 按鈕 hover 浮起 + 發光
- 圖表淡入效果
- Loading 流動漸變

### 配色
- 深色漸變背景 (灰 → 藍 → 紫)
- 藍紫漸變主色調
- 發光效果 (藍/紫/粉)

## 🚀 使用

```bash
cd frontend
yarn dev
```

訪問 http://localhost:3000

## ✨ 核心樣式類

```css
.glass           /* 半透明玻璃 */
.btn-glass       /* 玻璃按鈕 */
.card-glass      /* 可hover卡片 */
.metric-card     /* 指標卡片 (有光澤) */
.input-glass     /* 玻璃輸入框 */
.btn-primary     /* 主按鈕 (漸變) */
```

設計靈感來自 Apple 的軟硬體整合哲學！
